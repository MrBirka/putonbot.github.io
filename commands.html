Команда Использование Описание Права Псевдонимы
!сервер !сервер Информация об сервере (участников, ботов, каналов, статусы, владелец, уровень проверки, бусты, ID) Нет server
!очистка !очистка <количество> <участник> Очистка сообщений в данном канале (все или только у указанного пользователя) Управлять сообщениями clear, clr
!пригласить !пригласить Ссылка для приглошения бота на сервер Нет invite
!бан !бан <участник> [причина] Забанить участника на сервере Банить участников ban
!лс !лс <участник> [сообщение] Написать личное сообщение участнику Нет pm
!обьявление !обьявление [сообщение] Делает объявление на весь севрер Управлять сервером broadcast
!стат !стат Показывает статистику ресурсов (ОЗУ, CPU), версию бота и количество серверов, на котором находится бот Нет stats
!пинг !пинг Показывает скорость отправки данных на сервер Нет ping
!партнёры !партнёры Сообщение для канала "Партнёры" Нет partners
!роль !роль <действие> <участник> <роль> Выдаёт или забирает у участника роль Администратор role
!юзер !юзер [участник] Посмотреть статистику участника Нет user
!слоумод !слоумод <слоумод (в секундах> Изменяет задержку на сообщения в данном канале Управлять каналами slowmode
!кик !кик <участник> [причина] Кикает участника с сервера Кикать участников kick
